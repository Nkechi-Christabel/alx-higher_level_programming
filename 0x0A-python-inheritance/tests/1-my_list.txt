# Test cases for MyList class
===========================

The 1-my_list module defines a class MyList that inherits from list.

Test cases
==========

Import "MyList" from the "1-my_list" module:
>>> MyList = __import__('1-my_list').MyList

# Test 1: Printing an empty list in sorted order
>>> my_list = MyList()
>>> my_list.print_sorted()
[]

# Test 2: Printing a sorted list
>>> my_list = MyList()
>>> my_list.extend([1, 2, 3, 4, 5])
>>> my_list.print_sorted()
[1, 2, 3, 4, 5]

# Test 4: Printing a list with repeated elements
>>> my_list = MyList()
>>> my_list.extend([3, 3, 3, 3, 3])
>>> my_list.print_sorted()
[3, 3, 3, 3, 3]

# Test 5: Printing a list with negative numbers
>>> my_list = MyList()
>>> my_list.extend([-5, -2, 0, -3, -1])
>>> my_list.print_sorted()
[-5, -3, -2, -1, 0]
